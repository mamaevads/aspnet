Сайт должен быть на фреймворке aspnetcore. Главное, чтобы в нем было сдающее:

Для страниц использовать движок Razor

Регистрация пользователей, разные роли пользователей (администратор, модератор и т.д.)

Администратор должен иметь возможность создавать и редактировать материалы на сайте (Если это магазин, то материалами будут товары)

Загрузка картинок либо других файлов на сайт

Возможность комментировать материалы (Достаточно будет простого ввода теста, не обязательно делать мощный редактор для форматирования теста, цитирование и т.д.)

База данных, работа с базой по средствам EntityFrameworkCore(Использовать подход code first)

Внешний вид сайта в целом не важен. Можно использовать bootstrap или другой фреймворк.

Наличие JSтак же не обязательно, но желательно сделать хотя бы один ajaxзапрос и apicontroller для него

Сайт не должен быть уязвим к XSSатакам и SQLинфекциям (Если использовать Razorи EF правильно, то само получится)

 

В процессе работы нужно использовать git, для того что бы можно было видеть работу
